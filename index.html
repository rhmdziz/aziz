Project

Data Visualization of Covid 19 Indonesia

April 2023

This project is a simple web application built using the Streamlit framework and Pandas library. This application aims to analyze Covid-19 data in Indonesia by utilizing datasets from Kaggle which are then processed and visualized through graphs and tables. The available features are to display the growth of new cases, deaths and recoveries in Indonesia and in each province in Indonesia in the form of a line chart for a certain period. In addition, there are also metrics to display total cases, deaths and recoveries. This project is suitable for use by the general public or parties interested in analyzing Covid-19 data in Indonesia quickly and easily.
